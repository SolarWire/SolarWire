# SolarWire 项目计划

本目录包含 SolarWire 项目按版本号分类的计划文档。

## 版本列表

- [v1.0.0](./v1.0.0.md) - 初始发布版本 (2026-03-17)
- [v1.1.0](./v1.1.0.md) - 改进版本 (2026-03-20)
- [v2.0.0](./v2.0.0.md) - 规划中的重大更新

## 版本说明

### v1.0.x - 核心功能
- 完整的 DSL 语法和解析
- SVG 渲染器
- VSCode 扩展
- 基础 Note 系统

### v1.1.x - 体验优化
- 更好的文本换行
- 响应式预览
- 改进的 Note 卡片
- 代码清理和优化

### v2.0.x - 架构改进
- 在线编辑器
- 协作功能
- 插件系统

# SolarWire AI Prompt - Mobile App

这是移动端应用场景的特定配置。

## 一、场景特点

移动端应用通常具有以下特点：
- 较窄的画布（375-430px）
- 垂直布局为主
- 底部导航栏常见
- 内容区域占满宽度
- 触摸友好的大尺寸按钮
- 上下滚动是主要导航方式

## 二、尺寸规范

### 画布尺寸
- 推荐宽度：375-430px（iPhone 标准宽度）
- 高度根据内容自动调整

### 元素尺寸（触摸友好）
- 按钮最小尺寸：44x44px（苹果推荐）
- 按钮高度：44-56px
- 按钮宽度：占满可用宽度或最小 44px
- 输入框高度：44-52px
- 输入框宽度：占满可用宽度
- 文本最小字号：14px
- 标题字号：18-22px

### 间距规范
- 元素间距：8-16px
- 页面边距：16-24px

## 三、布局最佳实践

### 1. 常见布局结构

```solarwire
!title="Mobile App"
!size=14
!gap=12

// 容器矩形 - iPhone 12尺寸
[?] @(0,0) w=390 h=844 bg=#fff hide

// 顶部状态栏/导航栏
["<"] @(16,44) w=44 h=44 note="返回上一页"
"页面标题" @(68,58) size=18

// 主内容区
"欢迎回来" @(24,120) size=24 bold
"用户名" @(24,180)
["请输入用户名"] @(24,205) w=342 h=48 bg=#fff b=#ddd note="输入框。
- 支持手机号或邮箱登录
- 自动去除首尾空格
- 格式校验：手机号11位数字，邮箱需含@和域名
- 错误提示：格式错误时下方显示「请输入正确的手机号或邮箱」
- 最大长度：50字符"

"密码" @(24,270)
["请输入密码"] @(24,295) w=342 h=48 bg=#fff b=#ddd note="密码输入框。
- 密码显示为圆点
- 右侧有显示/隐藏切换图标
- 最小长度：6字符，最大：32字符"

["登录"] @(24,370) w=342 h=52 bg=#4CAF50 c=white note="提交登录。
- 成功：跳转首页，保存登录状态
- 失败：显示「用户名或密码错误」弹窗，清空密码框
- 按钮禁用条件：用户名或密码为空
- 防重复：点击后禁用3秒，或请求返回后恢复"

"忘记密码？" @(145,442) c=#2196F3 note="点击跳转到密码找回页面"

// 底部导航栏
["首页"] @(0,500) w=97 h=60 bg=#f5f5f5 note="切换到首页"
["发现"] @(98,500) w=97 h=60 bg=#f5f5f5 note="切换到发现页"
["消息"] @(195,500) w=97 h=60 bg=#f5f5f5 note="切换到消息页"
["我的"] @(293,500) w=97 h=60 bg=#f5f5f5 note="切换到个人中心"
```

### 2. 顶部导航栏
- 通常放在页面顶部
- 返回按钮在左侧 `@(16,44)`，标题在中间，操作按钮在右侧
- 高度：56-64px
- 触摸元素至少 44x44px

### 3. 表单布局
- 标签 + 输入框组合，垂直排列
- 按钮占满宽度，放在底部
- 输入框高度至少 44px

### 4. 卡片布局
- 使用圆角矩形 `()` 作为卡片
- 卡片宽度占满可用宽度
- 卡片内元素用绝对坐标定位
- 卡片之间保持适当间距

### 5. 列表布局
- 每个列表项至少 44px 高度
- 垂直排列，用绝对坐标
- 可以有左侧图标、中间文本、右侧箭头/开关

## 四、常见模式

### 登录页面
- 垂直居中布局
- Logo + 标题 + 表单 + 按钮（全宽）

### 列表页面
- 顶部搜索栏
- 列表项（垂直排列，每个至少 44px 高）
- 下拉刷新/上拉加载

### 详情页面
- 顶部返回按钮 + 标题
- 详情内容（垂直滚动）
- 底部操作按钮（全宽）

### 个人中心
- 顶部用户信息区域
- 设置列表
- 底部退出登录按钮

## 五、页面组织规则

**每个 SolarWire 代码块只处理一个独立视图：**

| 情况 | 处理方式 |
|------|---------|
| 弹窗/对话框 | 单独一个 SolarWire 片段 |
| 页面不同状态 | 每个状态单独一个片段 |
| 标签页切换 | 每个标签页单独一个片段 |

## 六、Note 写作规范

**禁止在 Note 中写的内容：**
- ❌ 视觉细节：颜色、字体、圆角、阴影、动画效果
- ❌ 技术实现: 接口名、数据库字段、加密方式
- ❌ 尺寸数值: 宽度、高度、间距

**Note 应该包含:**
- 功能行为: 点击/操作后发生什么
- 成功/失败场景: 不同结果的处理方式
- 输入限制: 格式、长度、必填等
- 边界条件: 空值、超长、特殊字符等
- 状态变化: 禁用条件、加载状态等
- 错误提示: 什么情况显示什么提示

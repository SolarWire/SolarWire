# SolarWire AI Prompt - Web Client
这是客户端 Web 场景的特定配置。
## 一、场景特点
客户端 Web 应用通常具有以下特点：
- 较宽的画布（1200-1440px）
- 水平布局为主
- 顶部导航栏常见
- 内容区域居中或左对齐
- 按钮和输入框尺寸适中
## 二、尺寸规范
### 画布尺寸
- 推荐宽度：1200-1440px
- 高度根据内容自动调整
### 元素尺寸
- 按钮高度：36-48px
- 按钮宽度：根据内容，最小 80px
- 输入框高度：36-44px
- 输入框宽度：200-400px（根据用途）
- 文本字号：12-16px
- 标题字号：18-24px
### 间距规范
- 元素间距：12-24px
- 页面边距：24-48px
## 三、布局最佳实践
### 1. 常见布局结构
```solarwire
!title="Web Client"
!size=14
!gap=16

// 容器矩形 - Web客户端
[?] @(0,0) w=1440 h=900 bg=#fff hide

// 顶部导航栏
["Logo"] @(50,0) w=120 h=60
["首页"] @(200,18) note="切换到首页"
["产品"] @(280,18) note="切换到产品页"
["关于"] @(360,18) note="切换到关于页"

// 主内容区
"页面标题" @(50,100) size=20 bold
("欢迎使用") @(50,140) w=1300 h=200 note="欢迎区域卡片"

// 侧边栏（可选）
["菜单项1"] @(50,360) w=200 h=40 note="点击切换到菜单项1"
["菜单项2"] @(50,412) w=200 h=40 note="点击切换到菜单项2"

// 主内容
["内容..."] @(274,360) w=1000 h=400 note="主要内容区域"
```

### 2. 导航栏
- 通常放在页面顶部
- Logo 在左侧 `@(50,0)`，导航项在中间或右侧
- 高度：56-64px

### 3. 表单布局
- 标签 + 输入框组合
- 按钮放在底部，右对齐或居中

### 4. 卡片布局
- 使用圆角矩形 `()` 作为卡片
- 卡片内元素用绝对坐标定位
- 卡片之间保持适当间距

## 四、常见模式

### 登录页面
- 居中布局
- Logo + 标题 + 表单 + 按钮

### 列表页面
- 顶部搜索/筛选栏
- 数据表格
- 分页控件

### 详情页面
- 面包屑导航
- 标题区域
- 详情内容（可能分多个卡片）
- 操作按钮

### 表单页面
- 标题
- 表单项（垂直排列）
- 提交/取消按钮

## 五、页面组织规则

**每个 SolarWire 代码块只处理一个独立视图：**

| 情况 | 处理方式 |
|------|---------|
| 弹窗/对话框 | 单独一个 SolarWire 片段 |
| 页面不同状态 | 每个状态单独一个片段 |
| 标签页切换 | 每个标签页单独一个片段 |

## 六、Note 写作规范

**禁止在 Note 中写的内容：**
- ❌ 视觉细节：颜色、字体、圆角、阴影、动画效果
- ❌ 技术实现: 接口名、数据库字段、加密方式
- ❌ 尺寸数值: 宽度、高度、间距

**Note 应该包含:**
- 功能行为: 点击/操作后发生什么
- 成功/失败场景: 不同结果的处理方式
- 输入限制: 格式、长度、必填等
- 边界条件: 空值、超长、特殊字符等
- 状态变化: 禁用条件、加载状态等
- 错误提示: 什么情况显示什么提示

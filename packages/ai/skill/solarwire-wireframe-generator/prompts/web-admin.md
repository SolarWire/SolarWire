# SolarWire AI Prompt - Web Admin
这是后台管理系统场景的特定配置。
## 一、场景特点
后台管理系统通常具有以下特点：
- 很宽的画布（1440-1920px）
- 左侧固定侧边栏
- 顶部导航/面包屑
- 数据密集（表格、图表、统计卡片）
- 多列布局
- 功能操作按钮多（新增、编辑、删除、导出等）
## 二、尺寸规范
### 画布尺寸
- 推荐宽度：1440-1920px
- 高度根据内容自动调整
### 元素尺寸
- 按钮高度：32-40px
- 按钮宽度：根据内容，最小 64px
- 输入框高度：32-36px
- 输入框宽度：160-320px（根据用途）
- 表格行高：40-48px
- 侧边栏宽度：200-280px
- 文本字号：12-14px
- 标题字号：16-20px
### 间距规范
- 元素间距：16-32px
- 页面边距：24-32px
- 表格单元格 padding：8-12px

### 2. 侧边栏
- 固定在页面左侧 `@(0,0)`
- Logo 在顶部，菜单项垂直排列
- 宽度：200-280px
- 菜单项高度：40-48px

### 3. 顶部栏
- 固定在内容区顶部
- 面包屑在左侧，操作按钮在右侧
- 高度：56-64px

### 4. 统计卡片
- 使用圆角矩形 `()`
- 水平排列，用绝对坐标
- 包含标题和数字
- 尺寸：200-300px 宽，100-140px 高

### 5. 数据表格
- 是后台管理系统的核心
- 宽度占满可用空间
- 表头背景色区分
- 操作列在最右侧
- 行高：40-48px

### 6. 表单布局
- 两列或三列布局，用绝对坐标
- 标签 + 输入框组合
- 重要字段在左上
- 按钮放在底部，右对齐

## 四、常见模式

### 用户列表页面
- 顶部搜索/筛选栏
- 批量操作按钮
- 数据表格（多选、操作列）
- 分页控件

### 数据统计页面
- 多个统计卡片
- 图表占位符
- 时间范围选择器

### 表单编辑页面
- 面包屑导航
- 表单（多列布局）
- 保存/取消按钮

### 设置页面
- 分组的设置项
- 开关、下拉框、输入框组合
- 保存按钮

## 五、页面组织规则

**每个 SolarWire 代码块只处理一个独立视图：**

| 情况 | 处理方式 |
|------|---------|
| 弹窗/对话框 | 单独一个 SolarWire 片段 |
| 页面不同状态 | 每个状态单独一个片段 |
| 标签页切换 | 每个标签页单独一个片段 |

## 六、Note 写作规范

**禁止在 Note 中写的内容：**
- ❌ 视觉细节：颜色、字体、圆角、阴影、动画效果
- ❌ 技术实现: 接口名、数据库字段、加密方式
- ❌ 尺寸数值: 宽度、高度、间距

**Note 应该包含:**
- 功能行为: 点击/操作后发生什么
- 成功/失败场景: 不同结果的处理方式
- 输入限制: 格式、长度、必填等
- 边界条件: 空值、超长、特殊字符等
- 状态变化: 禁用条件、加载状态等
- 错误提示: 什么情况显示什么提示

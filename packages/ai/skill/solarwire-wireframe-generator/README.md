# SolarWire 线框图生成器 Skill

这是一个完整的、可直接拷贝使用的 SolarWire 线框图生成器 Skill 包。

## 目录结构

```
solarwire-wireframe-generator/
├── SKILL.md             # Skill 主定义文件
├── README.md            # 说明文档（本文件）
└── prompts/             # Prompt 文件目录
    ├── mobile-app.md    # 移动端 App 场景
    ├── web-client.md    # 客户端 Web 场景
    └── web-admin.md     # 后台管理系统场景
```

## 功能特性

- 多轮对话，逐步确认需求
- 支持三种场景：移动端 App、客户端 Web、后台管理系统
- 输出完整的 .md 文档，包含：
  - 产品背景
  - 用户故事
  - 功能概述
  - 业务流程图（Mermaid）
  - 交互时序图（Mermaid）
  - SolarWire 线框图（多个页面/弹窗/状态）

## 预览方式

使用 VSCode SolarWire 插件实时预览 `.md` 文件中的 solarwire 代码块。

## 文件命名规范

- 主文档：`solarwire-[项目名称].md`

## 重要原则

1. **用户主动权**：不确定时一定要询问用户，给出建议让用户选择
2. **渐进式确认**：多轮对话，逐步确认，不要一次性生成太多
3. **灵活输出**：根据实际需求决定 .md 文档中包含哪些章节，不是所有章节都必须
4. **清晰反馈**：每一步都让用户知道在做什么

## 核心规范

1. **语法铁律**：所有元素必须有坐标，属性不用方括号
2. **元素选择**：按钮用矩形，标签用文本，图标才用占位符
3. **页面组织**：弹窗/状态/标签页必须用单独的 SolarWire 片段
4. **容器矩形**：每个页面必须有容器矩形，子元素不能超出父容器
5. **Note 规范**：详细但不写视觉/技术细节

## 支持的场景

### 移动端 App (mobile-app.md)
- 画布宽度：375-430px
- 垂直布局
- 触摸友好的大尺寸按钮
- 底部导航栏

### 客户端 Web (web-client.md)
- 画布宽度：1200-1440px
- 水平布局
- 顶部导航栏

### 后台管理系统 (web-admin.md)
- 画布宽度：1440-1920px
- 左侧侧边栏
- 数据表格、统计卡片

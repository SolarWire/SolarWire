# SolarWire AI Prompt - Web Client

这是客户端 Web 场景的特定配置。

## 一、场景特点

客户端 Web 应用通常具有以下特点：
- 较宽的画布（1200-1440px）
- 水平布局为主
- 顶部导航栏常见
- 内容区域居中或左对齐
- 按钮和输入框尺寸适中

## 二、尺寸规范

### 画布尺寸
- 推荐宽度：1200-1440px
- 高度根据内容自动调整

### 元素尺寸
- 按钮高度：36-48px
- 按钮宽度：根据内容，最小 80px
- 输入框高度：36-44px
- 输入框宽度：200-400px（根据用途）
- 文本字号：12-16px
- 标题字号：18-24px

### 间距规范
- 元素间距：12-24px
- 容器 gap：12-16px
- 页面边距：24-48px

## 三、布局最佳实践

### 1. 常见布局结构
```
{col} @(0,0) w=1400 gap=0
  // 顶部导航栏
  {row} gap=16 align=center
    ["Logo"]
    ["导航1"]
    ["导航2"]
    ["导航3"]
  
  // 主内容区
  {row} gap=24
    // 侧边栏（可选）
    {col} w=200 gap=8
      ["菜单项1"]
      ["菜单项2"]
    
    // 主内容
    {col} w=1000 gap=16
      ["页面标题"]
      ["内容..."]
```

### 2. 导航栏
- 通常放在页面顶部
- 使用水平容器 `{row}`
- Logo 在左侧，导航项在中间或右侧
- 高度：56-64px

### 3. 表单布局
- 使用垂直容器 `{col}` 组织表单项
- 标签 + 输入框组合
- 按钮放在底部，右对齐或居中

### 4. 卡片布局
- 使用圆角矩形 `()` 作为卡片
- 卡片内使用垂直容器组织内容
- 卡片之间保持适当间距

## 四、常见模式

### 登录页面
- 居中布局
- Logo + 标题 + 表单 + 按钮

### 列表页面
- 顶部搜索/筛选栏
- 数据表格
- 分页控件

### 详情页面
- 面包屑导航
- 标题区域
- 详情内容（可能分多个卡片）
- 操作按钮

### 表单页面
- 标题
- 表单项（垂直排列）
- 提交/取消按钮

## 五、示例结构

```
!title="Web Client Example"
!size=14
!gap=16

// PRD 内容区域
"产品背景：这是一个客户端 Web 应用" @(50,30) w=1300
"用户故事：作为用户，我想要..." @(50,60) w=1300

// 线框图开始
{col} @(50,100) w=1300 gap=0
  // 导航栏
  {row} gap=16 w=1300 h=60 bg=#f5f5f5 align=center
    ["My App"] @(50,0)
    ["首页"] @(+,0)
    ["产品"] @(+,0)
    ["关于"] @(+,0)
  
  // 主内容
  {row} @(0,+24) gap=24
    // 侧边栏
    {col} w=200 gap=8
      ["菜单1"] w=200 h=40
      ["菜单2"] w=200 h=40
      ["菜单3"] w=200 h=40
    
    // 内容区
    {col} w=1000 gap=16
      "页面标题" size=20 bold
      ["欢迎使用"] w=1000 h=200
```

# SolarWire AI Prompt - Mobile App

这是移动端应用场景的特定配置。

## 一、场景特点

移动端应用通常具有以下特点：
- 较窄的画布（375-430px）
- 垂直布局为主
- 底部导航栏常见
- 内容区域占满宽度
- 触摸友好的大尺寸按钮
- 上下滚动是主要导航方式

## 二、尺寸规范

### 画布尺寸
- 推荐宽度：375-430px（iPhone 标准宽度）
- 高度根据内容自动调整

### 元素尺寸（触摸友好）
- 按钮最小尺寸：44x44px（苹果推荐）
- 按钮高度：44-56px
- 按钮宽度：占满可用宽度或最小 44px
- 输入框高度：44-52px
- 输入框宽度：占满可用宽度
- 文本最小字号：14px
- 标题字号：18-22px

### 间距规范
- 元素间距：8-16px
- 容器 gap：8-12px
- 页面边距：16-24px

## 三、布局最佳实践

### 1. 常见布局结构
```
{col} @(0,0) w=375 gap=0
  // 顶部状态栏/导航栏
  {row} gap=8 w=375 h=56 bg=#f5f5f5 align=center
    ["<"] @(16,0)
    ["页面标题"] @(+,0)
    ["..."] @(R-16,0)
  
  // 主内容区（可滚动）
  {col} w=375 gap=12
    ["内容1"]
    ["内容2"]
    ["内容3"]
  
  // 底部导航栏
  {row} gap=0 w=375 h=60 bg=#f5f5f5 align=center
    ["首页"] w=93 h=60
    ["发现"] w=94 h=60
    ["消息"] w=94 h=60
    ["我的"] w=94 h=60
```

### 2. 顶部导航栏
- 通常放在页面顶部
- 使用水平容器 `{row}`
- 返回按钮在左侧，标题在中间，操作按钮在右侧
- 高度：56-64px
- 触摸元素至少 44x44px

### 3. 表单布局
- 使用垂直容器 `{col}` 组织表单项
- 标签 + 输入框组合，垂直排列
- 按钮占满宽度，放在底部
- 输入框高度至少 44px

### 4. 卡片布局
- 使用圆角矩形 `()` 作为卡片
- 卡片宽度占满可用宽度
- 卡片内使用垂直容器组织内容
- 卡片之间保持适当间距

### 5. 列表布局
- 每个列表项至少 44px 高度
- 垂直排列
- 可以有左侧图标、中间文本、右侧箭头/开关

## 四、常见模式

### 登录页面
- 垂直居中布局
- Logo + 标题 + 表单 + 按钮（全宽）

### 列表页面
- 顶部搜索栏
- 列表项（垂直排列，每个至少 44px 高）
- 下拉刷新/上拉加载

### 详情页面
- 顶部返回按钮 + 标题
- 详情内容（垂直滚动）
- 底部操作按钮（全宽）

### 个人中心
- 顶部用户信息区域
- 设置列表
- 底部退出登录按钮

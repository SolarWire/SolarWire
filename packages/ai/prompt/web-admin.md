# SolarWire AI Prompt - Web Admin

这是后台管理系统场景的特定配置。

## 一、场景特点

后台管理系统通常具有以下特点：

- 很宽的画布（1440-1920px）
- 左侧固定侧边栏
- 顶部导航/面包屑
- 数据密集（表格、图表、统计卡片）
- 多列布局
- 功能操作按钮多（新增、编辑、删除、导出等）

## 二、尺寸规范

### 画布尺寸

- 推荐宽度：1440-1920px
- 高度根据内容自动调整

### 元素尺寸

- 按钮高度：32-40px
- 按钮宽度：根据内容，最小 64px
- 输入框高度：32-36px
- 输入框宽度：160-320px（根据用途）
- 表格行高：40-48px
- 侧边栏宽度：200-280px
- 文本字号：12-14px
- 标题字号：16-20px

### 间距规范

- 元素间距：16-32px
- 容器 gap：12-20px
- 页面边距：24-32px
- 表格单元格 padding：8-12px

## 三、布局最佳实践

### 1. 常见布局结构

```
{row} @(0,0) w=1920 gap=0
  // 左侧侧边栏
  {col} w=240 h=full bg=#f5f5f5 gap=0
    ["Logo"] w=240 h=60
    ["菜单1"] w=240 h=44
    ["菜单2"] w=240 h=44
    ["菜单3"] w=240 h=44
  
  // 右侧主内容区
  {col} w=1680 gap=0
    // 顶部栏
    {row} w=1680 h=60 bg=#fff gap=16 align=center
      "面包屑 / 用户管理 / 列表" @(24,0)
      ["通知"] @(R-24,0)
      ["用户"] @(+,0)
    
    // 内容区域
    {col} @(0,+24) w=1680 gap=24
      // 统计卡片行
      {row} gap=24
        ("用户总数") w=280 h=120
        ("今日新增") w=280 h=120
        ("活跃用户") w=280 h=120
        ("收入") w=280 h=120
      
      // 数据表格
      {col} gap=16
        {row} gap=16 align=center
          ["搜索"] w=240 h=36
          ["筛选"] w=100 h=36
          ["新增用户"] @(R-24,0) w=120 h=36 bg=#4CAF50 c=white
        
        ## w=1632 h=400 border=1
```

### 2. 侧边栏

- 固定在页面左侧
- 使用垂直容器 `{col}`
- Logo 在顶部，菜单项垂直排列
- 宽度：200-280px
- 菜单项高度：40-48px

### 3. 顶部栏

- 固定在内容区顶部
- 使用水平容器 `{row}`
- 面包屑在左侧，操作按钮在右侧
- 高度：56-64px

### 4. 统计卡片

- 使用圆角矩形 `()`
- 水平排列，使用 `{row}`
- 包含标题和数字
- 尺寸：200-300px 宽，100-140px 高

### 5. 数据表格

- 是后台管理系统的核心
- 宽度占满可用空间
- 表头背景色区分
- 操作列在最右侧
- 行高：40-48px

### 6. 表单布局

- 两列或三列布局
- 标签 + 输入框组合
- 重要字段在左上
- 按钮放在底部，右对齐

## 四、常见模式

### 用户列表页面

- 顶部搜索/筛选栏
- 批量操作按钮
- 数据表格（多选、操作列）
- 分页控件

### 数据统计页面

- 多个统计卡片
- 图表占位符
- 时间范围选择器

### 表单编辑页面

- 面包屑导航
- 表单（多列布局）
- 保存/取消按钮

### 设置页面

- 分组的设置项
- 开关、下拉框、输入框组合
- 保存按钮
